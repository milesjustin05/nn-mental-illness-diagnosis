This is a application that uses meta-learning to potentially aid diagnoising mental illnesses. It works by training a generalized model, then for each test, trains it on two illnesses, and then does a binary classification between both illnesses.

How to run!

1. Download the files
2. Easist way to run is to use the Anaconda navigator to open VS code as to avoid a bunch of commands to activate a conda environment.
3. Once VS code is open, open the folder containing everything and run the model.py file.
